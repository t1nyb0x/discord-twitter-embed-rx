# クイックスタート: ダッシュボードバージョンのBotステータス表示

## 概要

Bot の Discord ステータスにダッシュボードのバージョンを表示する。
ダッシュボードが Redis にバージョンを書き込み、Bot がステータス更新時に読み取る。

## 変更対象ファイル

### 新規ファイル

| ファイル                           | 役割                                          |
| ---------------------------------- | --------------------------------------------- |
| `packages/shared/src/constants.ts` | Redis キー名・TTL・フォールバック値の共有定数 |

### 変更ファイル

| ファイル                       | 変更内容                                                                        |
| ------------------------------ | ------------------------------------------------------------------------------- |
| `packages/shared/src/index.ts` | `constants.ts` の re-export 追加                                                |
| `dashboard/src/startup.ts`     | 起動時にバージョン書き込み + TTL延長ジョブ追加                                  |
| `src/index.ts`                 | `updateStatus` を async 化し Redis からバージョン読み取り、表示フォーマット変更 |

## 表示フォーマット

- 正常時: `v2.1.0(Dashboard v1.2.0), 導入サーバー数: 5`
- フォールバック時: `v2.1.0(Dashboard 未接続), 導入サーバー数: 5`

## 動作タイミング

- ダッシュボード起動時: Redis に `app:dashboard:version` を SET（TTL 300秒）
- ダッシュボード稼働中: 2分ごとに TTL を 300秒に延長
- Bot 起動時 + 5分間隔: Redis から GET し、ステータスに反映

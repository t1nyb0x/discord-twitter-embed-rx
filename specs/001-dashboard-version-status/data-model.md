# データモデル: ダッシュボードバージョンのBotステータス表示

## エンティティ

### ダッシュボードバージョン情報

Redis に格納されるキー/バリューペア。

| 属性 | 型       | 説明                                          |
| ---- | -------- | --------------------------------------------- |
| キー | `string` | `app:dashboard:version`（固定）               |
| 値   | `string` | セマンティックバージョン文字列（例: `1.2.0`） |
| TTL  | `number` | 300秒（5分）                                  |

### バリデーションルール

- 値はセマンティックバージョン形式の文字列であること（`x.y.z`）
- 空文字・null・undefined の場合はフォールバック値を使用
- TTL は正の整数であること

### 状態遷移

```
[未設定] --(ダッシュボード起動)--> [バージョン設定済み（TTL付き）]
[バージョン設定済み] --(TTL延長)--> [バージョン設定済み（TTLリセット）]
[バージョン設定済み] --(ダッシュボード再起動)--> [バージョン更新（TTLリセット）]
[バージョン設定済み] --(TTL切れ)--> [未設定]
[未設定] --(Bot読み取り)--> フォールバック表示「未接続」
[バージョン設定済み] --(Bot読み取り)--> バージョン表示
```

## リレーションシップ

- **ダッシュボード → Redis**: 書き込み（SET + EXPIRE）・定期TTL延長（EXPIRE）
- **Bot → Redis**: 読み取り（GET）のみ
- **Bot → Discord**: ステータス表示（setActivity）
